# Git/GitHub勉強会2018

[ITCreateClub](https://itc.moe)主催のGit/GitHub勉強会2018で使用する資料です.  
資料の修正・改善等は[本リポジトリのissue](https://github.com/AfterSchoolStudy/git-study/issues)までお願いします.

## 環境構築

```bash
$ git clone https://github.com/tyokinuhata/git-study.git
$ cd git-study
$ gitbook serve
```

## ビルド

```
$ make build
```
