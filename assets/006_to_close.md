# その他のコマンド
ここまでいくつかのGitコマンドを解説してきたが, これらのコマンドはほんの一部に過ぎず, 他にも様々なコマンドが存在する.  
ここからのステップアップとしていくつかのコマンドを紹介しておく.

- git tag
- git rm
- git reset
- git revert
- git stash
- git cherry-pick
- git rebase
